App Highlights

This app contains:

  • A "Sync" button on the ActionBar that will connect to your ZSS Server and sync your local ZSS database with the MSSQL database on the server.
  • A full authentication dialog.
    (This version shows all of the various authentication fields for debugging purposes. In your production app, you may want to hard-code various fields and omit some of these fields from the dialog.)
  • A "Tables" button on the ActionBar of the Welcome Page that will show you a list of all of tables that were present in the database when this app was generated. Clicking on an individual table from this list will show you a roughly formatted list containing rows from the table. Clicking on a row from this list will show you a detailed view of that row in the database.
  • Insert and Update functionality to make changes in the app that can be Synced up to the server.
  • A Revert Changes button that will let you undo all changes that have been made in the app since the last Sync.

The source code illustrates the following concepts:

  • A functional ZumeroSync demo showing the use of the Zumero Client SDK for Xamarin.Forms on Android, iOS and Windows UWP using C# and Xamarin Studio.
  • Use of a custom class for each SQL table that knows about the table schema.
  • INSERT and UPDATE operations to modify data in the local SQLite database.
  • Custom column formatting routines that know about column datatypes.

The source code does not handle:

  • Database operations for DELETE.
  • Custom display widgets using column type information, such as a date/time picker.
  • Joins
